# رمز الخروج 127: الأمر غير موجود (Command Not Found)

### وصف مختصر
يُشير رمز الخروج 127 إلى أن الشل لم يجد الأمر المطلوب في PATH أو كأمر مدمج.

### الأسباب الشائعة لظهور الرمز
- خطأ إملائي في اسم الأمر.
- الأمر غير مثبت أو غير موجود في PATH.

### مثال عملي
```bash
nonexistentcommand  # أمر غير موجود
echo $?             # سيطبع 127
