# رمز الخروج 2: سوء استخدام التركيب (Misuse of Shell Builtins)

### وصف مختصر
يُشير رمز الخروج 2 إلى سوء استخدام لأوامر مدمجة في الشل، مثل استخدام خيارات غير صحيحة أو تركيب خاطئ.

### الأسباب الشائعة لظهور الرمز
- تمرير خيارات غير مدعومة إلى أمر مثل `ls`.
- خطأ في التركيب، مثل نسيان معاملات مطلوبة.

### مثال عملي
```bash
ls --invalid-option  # خيار غير صحيح
echo $?              # سيطبع 2
