# رمز الخروج 126: لا يمكن التنفيذ (Cannot Execute)

### وصف مختصر
يُشير رمز الخروج 126 إلى أن الأمر تم العثور عليه لكنه لا يمكن تنفيذه، غالبًا بسبب مشاكل في الأذونات.

### الأسباب الشائعة لظهور الرمز
- عدم وجود صلاحية تنفيذ على الملف (مثل chmod -x).
- ملف تالف أو غير قابل للتنفيذ.

### مثال عملي
```bash
chmod -x /bin/ls  # إزالة صلاحية التنفيذ
/bin/ls           # محاولة التنفيذ
echo $?           # سيطبع 126
