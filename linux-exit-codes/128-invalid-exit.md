# رمز الخروج 128: رمز خروج غير صالح (Invalid Exit)

### وصف مختصر
يُشير رمز الخروج 128 إلى محاولة استخدام قيمة خروج غير صالحة، غالبًا في سياقات برمجية أو إشارات.

### الأسباب الشائعة لظهور الرمز
- محاولة إنهاء برنامج بقيمة خارج النطاق (أكبر من 255 أو سالبة).
- خطأ في كتابة السكريبت عند استخدام `exit`.

### مثال عملي
```bash
exit 300  # قيمة خارج النطاق (0-255)
echo $?   # سيطبع 128 أو يُقتصر على 44 (300 % 256)
