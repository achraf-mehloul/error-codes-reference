# رموز الخروج (Exit Codes) في Linux/Unix

### مقدمة
في أنظمة Linux وUnix، يُعرف **رمز الخروج** (Exit Code) بأنه قيمة رقمية يقوم بها البرنامج أو الأمر عند انتهائه من التنفيذ. هذه الرموز تُستخدم للإشارة إلى حالة التنفيذ: نجاح أو فشل، وأحيانًا نوع الخطأ. يتراوح رمز الخروج عادةً بين 0 و255، حيث يُعتبر 0 نجاحًا، وأي قيمة أخرى تشير إلى خطأ.

### أهمية رموز الخروج
- **التحكم في السكريبتات**: تسمح للسكريبتات (مثل Bash) بالتحقق من نجاح الأوامر السابقة واتخاذ قرارات بناءً عليها (مثل استخدام `if` أو `&&`).
- **تصحيح الأخطاء**: تساعد المطورين في تحديد سبب فشل الأمر، مما يسهل التصحيح.
- **التوافقية**: معيار مشترك في معظم أنظمة Unix-like.

### كيفية التحقق من رمز الخروج
بعد تنفيذ أمر في الشل، يمكنك التحقق من رمز الخروجه باستخدام المتغير `$?` في Bash أو Zsh:
```bash
ls /nonexistent
echo $?  # سيطبع 2 (خطأ في الاستخدام)
